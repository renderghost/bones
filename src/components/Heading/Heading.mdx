# Heading Component

A customizable heading component that supports multiple levels (H1-H4) with consistent styling and optional truncation.

## Usage

```tsx
import { Heading } from '@bones/components';

function Example() {
	return <Heading level={1}>Hello World</Heading>;
}
```
